Welcome to 26quick!

In this game, your objective is to press all 26 keys on your physical keyboard in the order indicated by the on-screen keyboard. Just click the play area and press space to start!

I created this page first as a passion project and second as a way to get hands-on experience with frontend web development and exploratory data analysis. Contact info is in the footer. See the readme for more info.

Your current session data is saved to your browser's localStorage cache. localStorage is shared between all tabs and windows. The data does not expire and it persists between browser and OS restarts. See the readme for instructions on processing downloaded session data.
